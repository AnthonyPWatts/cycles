# Cycles Promo Audio Credits

The current 30-second master uses an original score and sound design generated by `tools/render_cycles_promo.py` specifically for Cycles. It does not contain third-party music or sound-effect recordings.

The soundtrack combines synthesised drones, pulses, transition sweeps, impacts, and a closing resolve at 48 kHz stereo. It is rendered with the picture so the command, battle, legacy, and final-title beats remain reproducible from the repository sources.

See [Promo Film Production Notes](PROMO-PRODUCTION.md) for the source frames, concept-image prompts, edit structure, and render command.
